# TrackMaker Lyrics Player
Currently displaying the lyrics of "Fashion center - TrackMaker" in your terminal with timing.

## Requirements
- Python 3.7+
- Terminal that supports ANSI escape codes (Linux/MacOS/Windows PowerShell)

## How to Run

```bash
git clone https://github.com/Ridhsuki/terminal-lyrics-lab.git
cd terminal-lyrics-lab/tracks/001-shitsunaikei-trackmaker
python3 index.py
```
